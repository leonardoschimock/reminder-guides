# Dev Notes Repository

This is not a real project.

This repository is a personal knowledge base made up entirely of `.md` files.  
It exists to store code snippets, Git commands, explanations, configurations, quick references, and development notes.

The purpose is simple:  
To serve as a memory extension for everyday development tasks.

There is no application, no production code, and no final product here — just organized documentation for learning and quick consultation.

If it helps someone else, even better 🚀

## What you might find here:
- Git commands and workflows
- Code snippets (various languages)
- IDE configurations
- Terminal commands
- Development reminders
- Useful links and references

Think of this as a developer’s second brain.
